# speak

OS X CLI for text-to-speech using `say`

## Requirements
* Node.js

## Installation

```
git clone git@github.com:furf/speak.git
cd speak
./install
```

## Usage

Basic usage:

```
speak "Hello, world"
```

For more:

```
speak -h
```
